# 美甲时光 - 微信小程序

## 项目简介

为美甲店打造的微信小程序，包含以下功能：

- 📅 **预约系统** - 客户在线预约美甲服务
- 👥 **客户管理** - 记录客户信息和喜好偏好
- 📦 **库存管理** - 管理美甲产品库存
- 📊 **财务统计** - 查看收入、利润、趋势分析

## 项目结构

```
nail-salon-miniapp/
├── app.js                    # 应用入口
├── app.json                  # 全局配置
├── app.wxss                  # 全局样式
├── sitemap.json              # 站点地图
├── images/                   # 图标资源（需自行添加）
├── utils/
│   └── storage.js            # 数据存储工具
└── pages/
    ├── index/                # 首页
    ├── booking/              # 预约页面
    ├── customers/            # 客户列表
    ├── customer-detail/      # 客户详情
    ├── add-customer/         # 添加客户
    ├── inventory/            # 库存管理
    ├── add-inventory/        # 添加库存
    ├── statistics/           # 财务统计
    └── profile/              # 个人中心
```

## 使用说明

### 1. 准备图标

在 `images/` 目录下添加以下图标文件：

- `home.png` / `home-active.png` - 首页图标
- `booking.png` / `booking-active.png` - 预约图标
- `customer.png` / `customer-active.png` - 客户图标
- `inventory.png` / `inventory-active.png` - 库存图标
- `stats.png` / `stats-active.png` - 统计图标

建议使用 81x81 像素的 PNG 图标，普通状态用灰色，选中状态用粉色(#E8A0BF)。

### 2. 导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目目录
4. 填入你的 AppID（或使用测试号）

### 3. 功能说明

#### 首页
- 显示今日概览（预约数、收入、客户数）
- 快捷操作入口
- 最近预约列表
- 库存预警提醒

#### 预约管理
- 新建预约（选择客户、服务、美甲师、时间）
- 按日期筛选预约
- 确认/取消/完成预约
- 完成服务自动更新库存和财务数据

#### 客户管理
- 添加客户（姓名、手机号、喜好偏好）
- 搜索客户
- 查看客户详情和消费记录
- 编辑/删除客户

#### 库存管理
- 添加库存物品
- 按分类筛选
- 库存预警提醒
- 服务完成自动扣减库存

#### 财务统计
- 按时间段查看（今日/本周/本月/全部）
- 收入、利润、利润率分析
- 热门服务 TOP5
- 每日收入趋势图

## 数据存储

本项目使用微信小程序的本地存储（`wx.setStorageSync`）模拟数据库，数据保存在用户设备上。

### 数据结构

- **customers** - 客户列表
- **bookings** - 预约列表
- **inventory** - 库存列表
- **transactions** - 交易记录
- **services** - 服务项目
- **staff** - 美甲师列表

## 主题定制

主题色定义在 `app.wxss` 中的 CSS 变量：

```css
--primary: #E8A0BF;        /* 主色-粉色 */
--primary-light: #F5D5E0;  /* 浅粉色 */
--primary-dark: #D4789F;   /* 深粉色 */
--gold: #C9A96E;           /* 金色 */
```
